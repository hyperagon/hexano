# Hexano
A simple hexagonal piano.

- Vibe coded but it required me to correct several things
- Everything is packed into a single *HTML* file
- Let's you pick different kinds of Wave
- Records the output into a *WAV* file
- Replays what was recorded like a piano roll
- Can Import and Export a melody to a *JSON* file

### Examples ([emoji from here](https://getemoji.com/#symbols))
- [Keyboard Cat](https://www.youtube.com/watch?v=J---aiyznGQ): 游댮游릭游릮游릭游댮游릭游릮游릭...
- Hexano : 游댮游리游릭游댯游멇릮 游멇릮游댯游릭游릮游리
- Censor: B5 (it's actually 1000Hz but this is the closest)
